## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test (not implemented yet)

```bash
# first build
$ docker-compose build
# run app inside docker container
$ docker-compose up
```
